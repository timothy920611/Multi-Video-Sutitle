
# 📄 JW 字幕批次提取工具 (JW Subtitle Batch Extractor)

這是一個免安裝、純網頁版的實用工具，專門用來批次提取與整理 JW 影片的字幕。
透過本工具，您可以輕鬆將多部影片的字幕抓取下來，系統會自動修復原始資料中的亂碼（如 `\r\n`、`""` 等），並排版成帶有時間軸、且易於閱讀的格式（如 PDF）。

🌐 **線上直接使用：** [JW 字幕批次提取工具](https://www.google.com/url?sa=E&source=gmail&q=https://timothy920611.github.io/Multi-Video-Sutitle/)

## ✨ 核心特色

- 🚀 **免安裝直接用**：完全在瀏覽器內執行，無需安裝任何軟體或設定環境。
- 📦 **批次處理能力**：支援一次貼上多個 JW 影片網址，自動依序處理。
- 🧹 **智慧文字清理**：自動過濾原始字幕檔中常見的程式符號（如 `","`、`\r\n`、多餘的引號等）。
- ⏱️ **精準時間軸**：完美保留 `[分:秒]` 的時間標記，方便回頭對照影片。
- 📑 **自動生成目錄**：匯出的報告首頁會自動建立「影片目錄」，多部影片也能一目了然。

## 📖 使用教學 (Step-by-Step)

### 步驟 1：開啟網頁

請點擊連結前往工具網站：[https://timothy920611.github.io/Multi-Video-Sutitle/](https://www.google.com/url?sa=E&source=gmail&q=https://timothy920611.github.io/Multi-Video-Sutitle/)

### 步驟 2：文字與排版設定 ⚙️

在開始提取之前，您可以根據需求在網頁介面上調整**文字設定**（依網站實際提供的選項為準）：

- **時間軸顯示**：選擇是否在每句字幕前保留 `[00:00:00]` 格式的時間標籤。
- **亂碼清理 (Clean Text)**：建議**預設開啟**。此功能會將原始資料庫格式（如 `"[00:00:23]\n","上個月我們觀看..."`）清理並轉換為乾淨的純文字。
- **輸出格式**：選擇您想要的匯出格式（例如：PDF 報告、純文字 TXT 或是 Markdown）。

### 步驟 3：輸入影片網址

在網頁的輸入框中，貼上您想要提取字幕的 JW 影片網址。
**注意：** 若要批次提取，請確保**每一行貼上一個網址**（按下 Enter 換行）。

範例：

```
https://www.jw.org/zh-hant/library/videos/.../video1
https://www.jw.org/zh-hant/library/videos/.../video2
https://www.jw.org/zh-hant/library/videos/.../video3
```

### 步驟 4：執行與下載

1. 點擊 **「開始提取」**（或對應的執行按鈕）。
2. 系統會在背景自動處理所有網址，請耐心等候進度條完成。
3. 處理完成後，點擊 **「下載報告」**，即可取得整理好的字幕檔案。

## ❓ 常見問題 (FAQ)

**Q1: 為什麼提取出來的文字還是有奇怪的引號跟換行符號？**
A: 請確保在「文字設定」中勾選了「清理亂碼」或「格式化字幕」的選項。工具的演算法會自動處理這些來自原始 CSV/JSON 的特殊字元。

**Q2: 支援哪些影片？**
A: 目前支援 JW.org 官方網站上的影片連結。只要該影片頁面本身有提供字幕，工具就能成功提取。
