🎬 JW 字幕批次提取神器 (動態時間開關版)

這是一個專為提取 JW 影片字幕所設計的輕量化網頁工具。只需貼上影片網址，就能自動抓取、清理並排版字幕。
最新的**「動態時間開關版」**加入了即時切換時間戳記的功能，讓您能自由選擇要「對照影片」還是「純文字閱讀」，並且匯出的所有檔案都會完美同步您的選擇！

✨ 核心特色

🚀 極速批次處理：支援一次貼上多個影片網址（最多 50 個），一鍵全自動依序提取。

⏱️ 動態時間開關 (全新功能)：

開啟時：保留 [00:01:23] 格式的時間戳記，適合對照影片畫面。

關閉時：瞬間隱藏所有時間，變為乾淨順暢的純文字文章，非常適合專心閱讀與列印。

🧹 深度淨化演算法：徹底解決原始字幕檔的亂碼問題！自動移除多餘的 HTML 標籤、引號、換行代碼（專治類似 "[00:00:23]\n"," 的頑固格式）。

🖨️ 原生列印優化：內建專屬的 Print CSS。按下 PDF 按鈕會自動隱藏網頁按鈕與操作介面，產生乾淨、無分頁裁切錯誤的高畫質向量 PDF。

🔒 純前端運作：所有資料處理都在您的瀏覽器中完成，沒有隱私外洩風險。

📝 如何使用

準備網址：複製您想要提取字幕的影片網址（支援 JW 官方格式）。

貼上並執行：在首頁的輸入框貼上網址（每行一個），點擊「開始批次提取」。

選擇閱讀模式：

在結果畫面的右上角，您可以隨時撥動「顯示時間」開關。

畫面會即時更新，展現帶有時間或純文字的排版。

匯出結果（畫面右下角的四個按鈕）：

📋 複製全部 (黑)：一鍵複製當前畫面的所有文字到剪貼簿。

📄 下載 TXT (藍)：將結果儲存為純文字檔。

🌐 下載 HTML (紫)：儲存為帶有目錄與跳轉功能的網頁檔。

🖨️ 完美 PDF (紅)：呼叫系統列印介面，請選擇「另存為 PDF」，即可獲得排版完美的閱讀手冊。

(註：您複製或下載的內容，會完全依照您當下「時間開關」的狀態來輸出！)

🛠️ 技術架構

本工具為單一 HTML 檔案，不需安裝任何伺服器環境，點開即可使用：

React 18：負責處理 UI 狀態與流暢的畫面切換。

Tailwind CSS：提供現代化、響應式的視覺設計與列印排版（@media print）。

CORS Proxy：內建備用代理伺服器機制，解決瀏覽器跨網域請求 (CORS) 限制。

⚠️ 注意事項

網路連線：由於跨網域限制，工具會透過公共代理伺服器 (CORS Proxy) 來抓取字幕。如果遇到「無法連線」的狀況，通常是代理伺服器暫時忙碌，請稍後再試。

網址格式：請確保貼上的是有效的單一影片網址，目前不支援整個節目的播放清單網址。
